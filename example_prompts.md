# Example Prompts

Use these prompts if you want to exercise the optional LLM layer or explain the design in the interview.

## Evidence extraction prompt

You are checking a resident progress note against structured falls-management rules. Return JSON only. For each requirement, identify a direct evidence snippet from the note or mark it absent. Do not invent missing details.

## Debrief prompt

Explain why the checker uses structured policy rules instead of passing the raw policy to the model, and describe how the system avoids false positives.
